Guida HTML + CSS per principianti

Impara a costruire un sito moderno da zero

HTML (HyperText Markup Language) è il linguaggio base per strutturare contenuti web. CSS (Cascading Style Sheets) serve a definirne l’aspetto.

Ecco un esempio base:

<!DOCTYPE html>
<html>
  <head><title>Titolo</title></head>
  <body>
    <h1>Ciao Mondo!</h1>
  </body>
</html>
